Subiendo nuestros archivos al espacio remoto

Nota: primero debemos de ingresar en GitHub

Al ingresar a Git debemos de crear un repositorio al cual, despues de crearlo nos mostrara en la parte superio derecha una opcion de code, al darle click se msotrara un enlace de nuestro repo git, utilizaremos la opcion https.

  • git remote add origin https://github.com/exmaple/example-repo.git
  • git pull origin master nos saldra un error al cual indicaremos el siguiente comando, pierde cuidado lo estas haciendo bien
  • git pull origin master --allow-unrealted-histories
  • git push origin master

Listo repo subida!!!

En GitHub vamos a poder realizar, pull request, forks , donde se lograran hacer merge contribuyendo con un repositorio que no sea el tuyo!!!

Para esto aprenderemos a realizar pull request

Primero es ser parte colaborador de un repositorio , posterio a eso tendras acceso a los datos del repositorio pudiendo clonarlos, al hacer cambios podras hacer pull y push, posterior a eso generar un pull request, donde se mostraran los cambios en el archivo, donde esperaras la respuesta de tu pull request, al tener un resultado aprobatorio, se genera un merge.

Haciendo un fork

Este paso es similar si embargo no perteneces al repositorio, puesto que deberias generar un fork que es clonar el repositorio en tu github, luego clonar tus archivos a tu espacio local, luego hacer cambios que creas convenientes, push!

Luego generar un pull request donde esperar respuesta del usuario dueño del repo clonado, asi haciendo un merge si el lo considera.
